Angular Frontend

Installing the project

Navigate to the Angular folder and run the following commands (on Linux).

  • $ npm install --save @angular/material
  • $ npm install --save @angular/material @angular/cdk
  • $ npm install --save bootstrap; npm install --save jquery
  • $ npm install (installs node modules (when in the Angular home folder))
  • compodoc
To generate the angular documentation, run "compodoc -p tsconfig.doc.json -s"

Components

The frontend is divided into several compontents.

Fragments

Located in Angular/src/app/fragments. The fragment component takes care of the fragments: retrieving them, combining the lines into fragments and displaying them.

Dashboard

Located in Angular/src/app/dashboard. The dashboard component takes care of user input. It allows for the creation, changing and deletion of fragments. Additionally, it allows changing passwords. Admins can create and delete users and change privileges.

Text

Located in Angular/src/app/text. The text components handles the retrieving and showing of full texts.

Api

Located in Angular/src/app/. The Api handles the communication between the components and the server.

Utilities

Located in Angular/src/app/. The utilities component provides functions used by all components.

Flask API

Installing the server

The installation of the backend is done using pip3 and the requirements file. In the server folder, run the following command:
  • $ pip3 install -r requirements.txt

Dependencies

The following Python packages are used for the backend:
  • numpy
  • flask
  • flask_cors
  • flask_restful
  • flask_jsonpify
  • couchdb
  • jsonpickle
  • fuzzywuzzy
