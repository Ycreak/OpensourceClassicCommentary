Quick Start

Modes of operation

The fragments on the left are clickable: when clicked, their commentary is shown in the right column. The panes are collapsable to keep things orderly. Primary editors can be selected: these are editors whose fragments have commentaries.

In Three Column Mode, a secondary editor can be selected: this editor does not necessarily have written commentaries. The order of fragments can be changed in this column, which is made possible using CSS.

Buttons

The change text button allows the user to change the author and select a new text. Authors are collected at startup, books are retrieved when selecting an author.

The bibliography button shows the bibliography of the selected book in a dialog. It separates Books, Articles and Websites.

The dahsboard button, if enabled, shows the dashboard in a dialogue. More about this component can be found in components or below.

The 3Columns button switches between three and two columns

The more button opens a menu to an About page and a possiblity to switch to the Text version of OSCC, which allows a complete text to have a commentary, instead of just fragments. As of now, only Seneca's Thyestes is added to this part.

Fragment Naming

Fragments can have numbers as name, as well as ranges of numbers. For example: Fragment 22 and Fragment 22-25 are both supported. Letters are supported, but won't be ordered correctly.

Fragments with multiple lines but without individual line numbering, for example TRF 134, should be entered per line with each line numbered starting from 1.
